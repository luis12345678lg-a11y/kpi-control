☁️ Base de Datos en la Nube
¿Cómo activar la base de datos?
1. Entra a supabase.com y crea una cuenta gratuita
2. Crea un nuevo proyecto
3. Ve a SQL Editor y ejecuta:
CREATE TABLE kpi_data (id TEXT PRIMARY KEY, data TEXT);
4. Ve a Settings → API y copia tu URL y anon key
Sin Supabase, los datos se guardan en este navegador y persisten al recargar la página. Con Supabase, se sincronizan en la nube y puedes abrirlos desde cualquier dispositivo.
